Cientista de Dados

O cientista de dados lida com o big data no dia a dia, ao coletar, gerenciar e modelar um grande volume de dados não-estruturados. Estes nada mais são do que informações que não estão organizadas de uma forma predefinida. O cientista de dados é essencial nos dias atuais, já que os dados são um dos recursos mais importantes do mundo moderno.

Essas competências permitem que ele consiga trabalhar com dados de diferentes formatos, na forma de tabelas, gráficos ou textos. Além disso, o cientista de dados também precisa ser capaz de interpretar os dados coletados e transformá-los em informações úteis para as empresas.

Conhecimentos necessários: Conhecimento em linguagens de programação de análise de dados (R, Python e SQL), infraestrutura de Big Data, Machine Learning, Data Mining e Álgebra Linear.

Níveis de Hierarquia: Junior (R$5.000), Pleno (R$8.710), Sênior (R$12.000)
